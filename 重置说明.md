# 🔄 如何重置抽奖系统

## 方法一：通过后台管理页面重置（推荐）

1. **访问后台管理页面**：
   - 在抽奖页面点击左上角 "📊 后台管理"
   - 或直接访问：`admin.html`

2. **清空所有数据**：
   - 点击 "🗑️ 清空所有数据" 按钮
   - 确认操作

3. **刷新抽奖主页面**：
   - 返回抽奖页面（`index.html`）
   - 按 `Cmd + R` 或 `F5` 刷新页面
   - 系统会自动重置，所有号码可以重新抽取

---

## 方法二：手动清空（如果后台不可用）

### 在浏览器控制台执行：

1. **打开浏览器开发者工具**：
   - 按 `F12` 或 `Cmd + Option + I`（Mac）
   - 或右键点击页面 → "检查"

2. **在 Console（控制台）中输入**：
   ```javascript
   localStorage.removeItem('lotteryData');
   location.reload();
   ```

3. **按回车执行**，页面会自动刷新并重置

---

## 方法三：清除浏览器数据

1. **打开浏览器设置**
2. **清除网站数据**：
   - Chrome: 设置 → 隐私和安全 → 清除浏览数据 → 选择 "Cookie 和其他网站数据"
   - Safari: 偏好设置 → 隐私 → 管理网站数据 → 删除
3. **刷新页面**

---

## ⚠️ 重要提示

- **清空数据后，所有已抽号码都会重置**
- **抽奖记录会全部删除**
- **建议在正式使用前先测试，然后清空数据重新开始**
- **清空后记得刷新主页面才能生效**

---

## 📝 正式使用前的建议流程

1. ✅ 测试抽奖功能
2. ✅ 测试后台管理
3. ✅ 确认一切正常
4. ✅ **在后台管理页面清空所有测试数据**
5. ✅ 刷新抽奖主页面
6. ✅ 开始正式抽奖
