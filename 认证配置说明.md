# GitHub 认证配置说明

## 方法一：使用 Personal Access Token（推荐）

### 步骤：

1. **创建 Personal Access Token**
   - 访问：https://github.com/settings/tokens
   - 点击 "Generate new token" → "Generate new token (classic)"
   - 填写 Note（备注）：例如 "HC Day Lottery"
   - 勾选权限：至少勾选 `repo`（完整仓库权限）
   - 点击 "Generate token"
   - **重要**：复制生成的 token（只显示一次！）

2. **使用 Token 推送**
   
   运行以下命令（会提示输入用户名和密码）：
   ```bash
   cd /Users/liuchangfei/HC-Day
   git push -u origin main
   ```
   
   - Username（用户名）：输入 `changfeiliu86-ux`
   - Password（密码）：**粘贴你的 Personal Access Token**（不是 GitHub 密码！）

## 方法二：使用 SSH（更安全，推荐长期使用）

### 步骤：

1. **生成 SSH 密钥**（如果还没有）
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"
   ```
   按回车使用默认路径，可以设置密码或直接回车

2. **复制公钥**
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
   复制输出的内容

3. **添加到 GitHub**
   - 访问：https://github.com/settings/keys
   - 点击 "New SSH key"
   - Title：填写备注，例如 "MacBook"
   - Key：粘贴刚才复制的公钥
   - 点击 "Add SSH key"

4. **修改 remote 为 SSH 地址**
   ```bash
   cd /Users/liuchangfei/HC-Day
   git remote set-url origin git@github.com:changfeiliu86-ux/hc-day-lottery.git
   git push -u origin main
   ```

## 方法三：安装 GitHub CLI（最简单）

```bash
# 安装 GitHub CLI
brew install gh

# 登录
gh auth login

# 然后就可以直接推送了
cd /Users/liuchangfei/HC-Day
git push -u origin main
```

## 推荐流程

1. **先确保 GitHub 仓库已创建**：
   - 访问：https://github.com/new
   - 仓库名：`hc-day-lottery`
   - 选择 Public
   - 创建仓库

2. **使用 Personal Access Token**（最快）：
   - 创建 Token：https://github.com/settings/tokens
   - 运行推送命令，使用 Token 作为密码

3. **或者安装 GitHub CLI**（最方便）：
   ```bash
   brew install gh
   gh auth login
   ```
