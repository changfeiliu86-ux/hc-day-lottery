# 🎉 HC Day 抽奖系统

一个精美的抽奖系统，支持11个等级的奖品，确保数字不重复，带有完整的后台管理功能。

## 功能特点

- ✨ 精美的抽奖动画效果
- 🎯 确保数字不重复
- 📊 完整的后台管理系统
- 🌐 支持中英文切换
- 📱 响应式设计，支持移动端
- ⌨️ 支持空格键快捷操作

## 文件说明

- `index.html` - 抽奖主页面
- `admin.html` - 后台管理页面
- `script.js` - 主页面逻辑
- `admin.js` - 后台管理逻辑
- `style.css` - 样式文件

## 使用方法

### 本地运行

```bash
# 启动本地服务器
python3 -m http.server 8000

# 访问
http://localhost:8000
```

### 部署

详见 [DEPLOY.md](./DEPLOY.md)

## 奖品配置

- **大奖**：充电宝（1001-1006）、HC 马克杯（981-1000）
- **中奖**：毛巾（931-980）、帆布袋（901-930）、银色行李牌（851-900）、笔（801-850）
- **小奖**：折叠杯（751-800）、塑料文件夹（451-750）、环保袋（401-450）、塑料文件夹（201-400）、纸质文件夹（1-200）

## 技术栈

- 纯 HTML/CSS/JavaScript
- LocalStorage 数据存储
- 无需后端服务器
